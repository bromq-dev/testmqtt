# 2.4 Payload

Some MQTT Control Packets contain a payload as the final part of the packet, as described in Chapter 3. In the case of the PUBLISH packet this is the Application Message. [Table 2.6 - Control Packets that contain a Payload](#_Table_2.6_-) lists the Control Packets that require a Payload.

##### Table 2.6 - Control Packets that contain a Payload

| **Control Packet** | **Payload** |
| ------------------ | ----------- |
| CONNECT            | Required    |
| CONNACK            | None        |
| PUBLISH            | Optional    |
| PUBACK             | None        |
| PUBREC             | None        |
| PUBREL             | None        |
| PUBCOMP            | None        |
| SUBSCRIBE          | Required    |
| SUBACK             | Required    |
| UNSUBSCRIBE        | Required    |
| UNSUBACK           | None        |
| PINGREQ            | None        |
| PINGRESP           | None        |
| DISCONNECT         | None        |
